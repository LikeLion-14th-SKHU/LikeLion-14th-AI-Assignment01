# LikeLion-14th-Assignment-Template
🦁 SKHU 멋쟁이사자처럼 14기 과제 PR 템플릿 레포지토리입니다.

1. 선택한 서비스
 댓글 부적절 탐지 서비스
2. 이 서비스는 무엇을 에측하는지
 댓글이 부적절한 내용을 포함하고 있는지를 예측하여 자동 분류한다.
3. 어떤 data가 사용될까
 댓글 텍스트 데이터, 욕설 및 혐오 표현 데이터, 정상/부적절 댓글 라벨 데이터, 신고 기록
4. 모델은 어떻게 학습할까?

(1) 텍스트 분류 모델: 댓글을 정상과 부적절 댓글로 분류한다.
  
   (2) 언어 패턴 분석 모델: 욕설, 혐오적 표현, 공격적 문장들의 패턴을 학습한다.
  
   (3) 자연어 처리 모델: 사용자의 문장과 문맥을 분석하여 적절/부적절 여부를 정확하게 판단한다.
  
5. 이 모델이 더 좋아지려면 어떻게 해야 할까?
     
   (1) 데이터 개선: 최신 신조어와 같은 다양한 말투와 표현들이 포함된 댓글의 데이터를 반영하여 학습시켜야 한다.
     
   (2) 모델 성능 향상: 문장 전체의 의미를 더 잘 파악할 수 있도록 모델의 성능을 향상시켜 정상적인 댓글을 차단하지 않도록 한다.
